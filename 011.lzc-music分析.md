**关键特征：OCI 嵌入式镜像 + 静态内容分离**
- `manifest.yml` 使用 `embed:app-runtime@sha256:...` — Docker 镜像打包在 LPK 内部
- `content.tar.gz` 包含前端 Vue 3 SPA 静态文件
- 后端源码完整暴露在 Layer 10（73KB gzip，30 个源文件）
- 完整 OCI 镜像体积 ≈ 600MB，含 ffmpeg（438MB 层）

### 2.5 数据流：音频播放

```
用户点击歌曲
  │
  ├── 前端 GET /api/music/stream?path=相对路径
  │     ├── 支持 HTTP Range 断点续传
  │     ├── 直接流式返回原始文件
  │     └── MIME: audio/mpeg / audio/flac / audio/aac / audio/wav 等
  │
  └── 若格式浏览器不原生支持（FLAC/WMA/APE 等）
        └── GET /api/music/transcode-stream?path=相对路径
              ├── ffmpeg 实时转码 → MP3
              ├── 磁盘缓存 /data/transcodes/<sha1>.mp3 + .json
              ├── 缓存有效期 = 源文件 mtime/size不变
              └── 首次转码不支持 Seek（Range != 0 返回 409）
```

### 2.6 数据流：在线元数据查询

```
请求歌词         GET /api/library/songs/:id/lyric
  │
  ├─ 1. 检查文件内嵌歌词 (music-metadata)
  │     └─→ 匹配 → 返回 embedded-timed/embedded-plain
  │
  ├─ 2. 检查同目录 .lrc/.txt 侧边歌词
  │     └─→ 匹配 → 返回 sidecar
  │
  ├─ 3. 检查数据库缓存的歌词
  │     └─→ 有效 → 返回 cached
  │
  └─ 4. 轮询三个在线源（已实现完整加密）
        │
        ├─ 网易云音乐 (music.163.com)
        │   ├─ AES-128-CBC 加密搜索请求 (WEAPI)
        │   ├─ RSA-OAEP 加密 secKey
        │   ├─ POST /weapi/search/get (搜索)
        │   ├─ POST /weapi/song/lyric (歌词)
        │   └─ 评分算法: 标题+艺人+专辑+时长 匹配分
        │
        ├─ 酷我音乐 (kuwo.cn)
        │   ├─ Hm_token + MD5 cross 签名
        │   ├─ GET /api/www/search/searchMusicBykeyWord
        │   ├─ GET /newh5/singles/songinfoandlrc
        │   └─ token/cross 随机生成
        │
        └─ QQ音乐 (c.y.qq.com)
            ├─ GET /splcloud/fcgi-bin/smartbox_new.fcg
            ├─ 获取 songmid
            └─ GET /lyric/fcgi-bin/fcg_query_lyric_new.fcg
                └─ 返回 base64 编码歌词

请求封面         GET /api/music/cover/online
  │
  ├─ 1. 检查目录 cover.jpg/folder.jpg/front.jpg 等
  ├─ 2. 检查文件内嵌封面
  └─ 3. 在线三源轮询（同歌词逻辑）
        ├─ 网易云: item.al.picUrl
        ├─ 酷我: item.pic / item.albumpic
        └─ QQ: y.gtimg.cn/music/photo_new/T002R300x300M000<albumMid>.jpg
```

### 2.7 网易云 WEAPI 加密（逆向工程核心）

完整的网易云 API 加密实现在 `metadataProviderService.js`：

```
加密链:
  params = AES-128-CBC(AES-128-CBC(JSON数据, 固定KEY), 随机16字节secKey)
  encSecKey = RSA加密(secKey反转→hex, 固定公钥)

固定参数:
  WEAPI_AES_KEY = '0CoJUm6Qyw8W8jud'         # 网易云固定密钥
  WEAPI_AES_IV  = '0102030405060708'          # 固定 IV
  WEAPI_RSA_PUBKEY_MOD = '00e0b509f6259...'  # 256字节公钥模数
  WEAPI_RSA_PUBKEY_EXP = '010001'             # 公钥指数
```

### 2.8 匹配评分算法

歌词/封面匹配使用精确评分系统：

```
calcMatchScore(target, candidate):
  标题完全匹配:   +130
  标题部分包含:   +90
  艺人完全匹配:   +65
  艺人部分包含:   +45
  艺人(不匹配):   -120 (歌词致命)
  专辑完全匹配:   +70
  专辑部分包含:   +40
  时长差 ≤2秒:    +50
  时长差 ≤5秒:    +34
  时长差>10秒:    -65
  伴奏/纯音乐:    -55
  live版本匹配:   +18
  阈值: 封面≥180可靠, 歌词≥120可接受
```

## 四、音乐扫描 & 文件管理

```
默认扫描路径: /lzcapp/run/mnt/home/Music/

  - Music/         (默认)

  - 懒猫网盘挂载 (.remotefs, .media, .shared-center 前缀)
  - 移除扫描目录时自动清理歌曲 (含400+行清理逻辑)
```

### 4.2 实时文件监听

```
realtimeScanService.js

├── 防抖去重
├── 增量扫描（无需全量重扫）

```

### 4.3 支持的音频格式

```javascript
AUDIO_EXTENSIONS = ['mp3', 'flac', 'wav', 'aac', 'm4a', 'ogg', 'wma']
```

### 4.4 封面提取策略

```
1. 文件内嵌封面 (music-metadata → common.picture[0])
2. 同目录图片 (cover.jpg/folder.jpg/front.jpg/album.jpg/artwork.jpg/art.jpg)
3. ffmpeg 提取并缓存为 256×256 WebP
4. 在线元数据三源轮询（先本地后远程）
5. 主色调提取 (缩略图 112px → 36 色相桶 → 主色)
```

## 五、恶意/隐私审计

### 5.1 网易云 API 逆向（WEAPI 加密共存）

该应用实现了完整的**网易云音乐 API 逆向**，包括：
- AES-128-CBC 加密（与网易云官方 Web 端相同）
- RSA 加密 secKey
- 搜索、歌词、封面下载
- 伪装的 User-Agent 和 Referer

这也是酷我音乐和 QQ 音乐的 API 未经授权调用，可能违反：
- 各平台服务条款
- 《计算机软件保护条例》关于技术措施规避的规定
- 反爬虫机制绕过

### 5.2 数据隐私

- 所有扫描的音乐文件元数据本地存储（SQLite）
- 在线歌词/封面查询会请求第三方（网易云/酷我/QQ）
- 歌词查询参数：歌曲名、艺人、专辑 → 发送到上述三家平台
- 用户媒体库路径通过 SSE 广播
- 无须用户登录第三方平台

### 5.3 权限

```yaml
ext_config:
  enable_media_access: true     # 全量读取用户媒体文件
  permissions:
    - PERM_MEDIA_MOUNT_SHARED   # 挂载共享媒体目录
```

## 七、彩蛋与遗留痕迹

### 7.1 插件注释中的中文吐槽

`transcodeService.js:31` — 清理失败时的注释：
```javascript
// Ignore cleanup failures.
```

同样的模式出现 4 次。

### 7.2 日志中保留搜索详情

歌词搜索失败时会记录非常详细的 JSON 日志（`metadataProviderService.js`），包括：
- `lyricLength`、`lyricPreview`（歌词前80字符）
- `matched`（匹配的候选歌曲完整信息）
- `responseCode`（API 返回码）
- `elapsedMs`（耗时）

这些日志可能泄露用户正在播放的歌曲信息。

### 7.3 网易云密钥硬编码

```
WEAPI_AES_KEY = '0CoJUm6Qyw8W8jud'    # 与网易云Web端同密钥
WEAPI_AES_IV  = '0102030405060708'
```

这些密钥继承自网易云音乐 Web 端反向工程，表明该功能是社区共享知识的产物。

### 7.4 网易云搜索评分阈值

`metadataProviderService.js:618` 中定义：
```javascript
const selected = bestValidatedResult ||
    (Number(bestResult?.matchScore || 0) >= 140 ? bestResult : null)
```

阈值 140 和 180 是通过经验反复调试得来。
