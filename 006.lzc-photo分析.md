## 一、LPK 包解构

### 1.1 归档结构

`cloud.lazycat.app.photo-v2.2.12.lpk` 是一个 POSIX tar 归档，内嵌 `content.tar` 作为应用主体。

```
$ tar -tf cloud.lazycat.app.photo-v2.2.12.lpk
content.tar              # 61MB — 照片应用主体
exports/
  mcp-providers/
    default/
      mcp.yml            # MCP 协议端点配置: endpoint: /api/mcp
icon.png                 # 191KB — 应用图标
manifest.yml             # 服务编排清单（核心架构描述文件）
package.yml              # 包元数据
```

### 编译信息

从二进制提取的 ldflags：
```
-ldflags="-X 'main.Version=2.2.12' -X 'main.Build=2026-06-18T01:28:30Z' -s -w"
```
`-s -w` 剥离了调试符号（DWARF），所以是 stripped 二进制。

## 九、文件传输协议

基于 TUS（Resumable Upload Protocol）的自定义实现：

```
请求头:
  Tus-Resumable: 1.0.0
  Upload-Offset: <bytes>
  Upload-Length: <total_bytes>
  Upload-Concat: partial/final
  Tus-Extension: creation,termination,concatenation
  Tus-Checksum-Algorithm: md5

响应头:
  Upload-Offset: <new_offset>
  Tus-Resumable: 1.0.0
```
