## 3. 入口脚本体系

### run.sh — 主入口

```
  run.sh
    │
    ├── 1. 数据目录迁移
    │      /lzcapp/var/wechat → /lzcapp/var/apps/wechat
    │
    ├── 2. 文件系统集成
    │      ├── 创建 shortcuts 目录
    │      ├── ln -s /lzcapp/clientfs         → shortcuts/本地保存
    │      ├── ln -s /lzcapp/document          → shortcuts/懒猫网盘
    │      └── ln -s /lzcapp/media             → shortcuts/外接磁盘
    │
    ├── 3. 桌面环境配置
    │      ├── GTK 书签: /lzcapp/pkg/content/gtk-bookmark
    │      ├── Qt 书签: /lzcapp/pkg/content/qt-bookmark.ini
    │      └── QT_QPA_PLATFORMTHEME=gtk3
    │
    ├── 4. xpra 环境变量
    │      XPRA_SCALING=false
    │      XPRA_DOWNSCALE=false
    │      XPRA_TEXT_USE_VIDEO=true
    │      XPRA_MIN_VIDEO_FPS=0
    │      XPRA_ENCODING_STRICT_MODE=true
    │      XPRA_DEFAULT_CONTENT_TYPE=video
    │      XPRA_CLIPBOARD_REMOTE_TIMEOUT=5000
    │
    ├── 5. xpra 配置安装
    │      ├── 90_lzc.conf → /tmp/catlink/xpra.conf
    │      ├── dummyscreen.sh → /usr/bin/spectacle
    │      └── patch/compress.py → xpra 的 window 模块
    │
    └── 6. 启动 xpra (无限循环)
           xpra seamless --bind-ws 127.0.0.1:7890 --daemon=no
```

```
  │
```

## 5. drop 二进制 (X11 拖放)

**文件信息：**
```
drop: ELF 64-bit LSB PIE executable, x86-64, dynamically linked,
      interpreter /lib64/ld-linux-x86-64.so.2,
      for GNU/Linux 3.2.0, NOT STRIPPED

源文件: drop.c
编译器: GCC (Ubuntu 15.2.0-16ubuntu1) 15.2.0
依赖: libX11.so.6, libc.so.6
```

**功能：** X11 拖放文件到远程窗口

```
drop <filename...>
  │
  ├── get_target_info()
  │   ├── 环境变量 TARGET_WINDOW_ID → X 窗口 ID
  │   └── XQueryPointer → 鼠标所在窗口
  │
  ├── is_target_xdnd_aware()
  │   XGetWindowProperty → 检查 XdndAware 协议
  │
  └── do_dnd()
      ├── XdndEnter           → 进入目标
      ├── XdndPosition        → 设置位置
      │   └── XSetSelectionOwner → text/uri-list (file://%s)
      ├── XdndDrop            → 执行放置
      └── send_selection_notify → 通知完成
```

流程：前端文件拖放 → rim `/dnd/drop` API → drop 二进制 → X11 Xdnd 协议 → xpra Seamless 窗口

## 6. xpra 远程显示配置

### 90_lzc.conf 核心配置

```ini
# === 输入法 ===
start-env=GTK_IM_MODULE=ibus
start-env=QT_IM_MODULE=ibus
start-env=XMODIFIERS="@im=xim"
input-method=none

# === 音频 ===
audio=yes
speaker=on
microphone=off
pulseaudio=yes

# === RIM 启动 ===
# rim 在 xpra 之后启动（依赖 session dbus）
start-env=GOLOG_LOG_LEVEL=warn
start=/lzcapp/pkg/content/rim

# === 默认应用 ===
start-child-after-connect=/lzcapp/pkg/content/start_default_app.sh
exit-with-children=yes
terminate-children=yes
```

### 显示编码配置

```ini
# === 编码器 ===
quality=100
video-encoders = all              # 所有视频编码器
csc-modules = libyuv              # 色彩空间转换
compressors = lz4, none           # 压缩算法 (禁用 brotli)
packet-encoders = rencodeplus, none

# === 虚拟显示器 ===
xvfb=Xdummy                       # 使用 Xdummy (多显示器支持)
bandwidth-limit=500Mbps
```

### window/compress.py 补丁

`patch/compress.py` (136KB) 来自 xpra 项目 (GPL v2) 的 `xpra/server/window/compress.py`，包含：

- 窗口压缩编码器选择逻辑
- 有损/无损编码切换
- 批量延迟计算
- RGB 重格式化
- 窗口图标源处理
- 性能统计
