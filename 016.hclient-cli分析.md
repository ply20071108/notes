## 1. CLI 命令系统

### 1.1 命令树

```
hclient-cli
├── tui                    # 交互式 TUI 界面（核心入口）
├── init-box               # 初始化新盒子（首次使用）
│   └── --address ADDR --pincode PINCODE --box-name NAME --admin USER [--password | --password-stdin]
├── login                  # 登录盒子
│   ├── qr                 # 扫码登录
│   │   └── --box-name NAME [--address ADDR]
│   ├── manual             # 手动输入密码登录
│   │   └── --box-name NAME [--address ADDR] [--password | --password-stdin]
│   └── tfa                # 双因素认证登录
│       └── --box-name NAME --tfa-code CODE [--address ADDR]
├── config                 # 配置管理
│   ├── set                # 设置配置项
│   └── login              # 配置登录凭据
├── daemon                 # 守护进程管理
│   └── (启动/停止 TUI 后台服务)
├── install                # 安装管理模式
│   ├── [--exec-path PATH] # 安装为托管服务
│   └── install-tun        # 安装 TUN 虚拟网卡
├── upgrade                # 升级盒子（远程升级 LCMD OS）
│   └── [--address ADDR]
├── kill                   # 终止守护进程
├── upload-log             # 上传调试日志到远程
│   └── [--address ADDR]
├── lzc-auth-token         # 查询 LZC 认证令牌
│   └── [--address ADDR]
└── stop-daemon            # 停止托管守护进程

全局标志: --address, --box-name, --password-stdin, --exec-path, --origin-server
环境变量: HCLIENT_CLI_CFG, HPORTAL_LOG_LEVEL, HPORTAL_CREATE_TUN, HPORTAL_CLI_TUI
```

### 1.3 数据流（以 `login manual` 为例）

```
main.runLoginManualCommand
  → main.openHeadlessRuntime
    → main.startPCClientRuntime
      → NewPortalClient (依赖注入)
        → grpc.Dial (连接到盒子 ShellAPI)
      → main.loginBoxTarget
  → PortalClient.LoginBox
    → connectBoxBackend.DoLogin
      → loginWithSRP
        → LoginHServerService/LoginWithSRP gRPC 调用
      → queryHServerInfo
      → hclientService 启动 (打洞/TFA/dialog)
  → main.waitForBoxLoginResult
    → 显示登录结果
```

## 5. 守护进程 (Daemon) 系统

### 5.1 daemon 生命周期

```
main.startDaemonProcess
  → 写入 daemon attach info (PID, socket path)
  → startPCClientRuntime (安装信号处理器)
  → daemonControl (gRPC 控制服务)
    → ShellCore 服务实现
    → UtilService 服务实现
  → 等待终止信号

管理方式:
├── systemd:  hclient-cli-systemd-*.service (系统/用户)
├── cron:     crontab 条目 (主要入口)
└── attatch:  daemonAttachInfoPath (PID 文件 + socket)

main.validateDaemonStartup → main.resolveDaemonEndpoint
main.waitForDaemonSocket → main.waitForDaemonStop
main.killDaemonProcesses → main.forceTerminateProcess
```

### 9.1 SRP 认证流程

```
LoginWithSRP 流程:
  客户端                                          盒子
    │                                               │
    ├── LoginWithSRPClient { I, A } ──────────────► │
    │                                               ├── LoginWithSRPServer { s, B }
    │◄──────────────────────────────────────────────┤
    │                                               │
    ├── (客户端计算 K = H(S), M = H(H(N) xor H(g), H(I), s, A, B, K))
    │                                               │
    ├── LoginWithSRPClient { M } ──────────────────► │
    │                                               ├── (验证 M, 计算 H(A, M, K))
    │◄───────────────────────────────────── 完成 ──┤
    │                                               │
    └── 安全会话建立 ──→ PunchHole / ServerChanged ──►
```
