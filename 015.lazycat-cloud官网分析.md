## 安全观察

| 发现 | 严重度 | 说明 |
|---|---|---|
| `/api/_nuxt_icon` 生产暴露 | 🟡 中 | Nuxt DevTools 功能未在生产环境禁用 |
| 开发环境域名残留 | 🟢 低 | `0.0.0.0`, `localhost`, `127.0.0.1` 留在生产配置 |
| 腾讯云 COS 存储桶泄露 | 🟢 低 | playground 图片存储桶 URL 硬编码 |
| 无 CSP 头 | 🟢 低 | 缺少内容安全策略 |
| HSTS 未启用 | 🟢 低 | HTTP 严格传输安全未配置 |
| Cookie 无 HttpOnly | 🟢 低 | 部分 Cookie 未设 HttpOnly 标志 |
